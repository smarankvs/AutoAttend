COMPLETE WORKFLOW:
================================================================================

STEP 1: Student Photo Upload
  → Student logs in
  → Goes to "My Profile"
  → Uploads clear face photo
  → System encodes face
  → Stored in database

STEP 2: Teacher Creates Class
  → Teacher creates class
  → Enrolls students
  → Class ready for attendance

STEP 3: Teacher Uploads Class Photo
  → Takes photo of classroom
  → Uploads via Attendance page
  → System processes image:
     • Detects all faces
     • Matches with student photos
     • Marks recognized as present
     • Marks others as absent
  → Shows results with confidence

STEP 4: View & Manage Attendance
  → Students view their attendance
  → Teachers view all students
  → Teachers can edit if needed
  → Real-time updates
